# ビルド修正

C#(.NET)のビルドおよび型/アナライザエラーを段階的に修正します:

1. ビルドを実行(この環境は ICU 無のためインバリアント指定):
   ```bash
   export DOTNET_ROOT="$HOME/.dotnet"; export PATH="$HOME/.dotnet:$PATH"
   export DOTNET_SYSTEM_GLOBALIZATION_INVARIANT=1
   dotnet build
   ```

2. エラー出力を解析:
   * ファイル/プロジェクト別にグループ化
   * エラーコード(CSxxxx)と重大度で並び替え

3. 各エラーについて:
   * エラーコンテキストを表示(前後 5 行)
   * 問題を説明(型不一致・未定義参照・Null 許容警告など)
   * 修正案を提案
   * 修正を適用
   * `dotnet build` を再実行
   * エラーが解決されたか確認

4. 以下の場合に停止:
   * 修正で新しいエラーが発生
   * 同じエラーが 3 回の試行後も続く
   * ユーザーが一時停止をリクエスト

5. サマリーを表示:
   * 修正されたエラー
   * 残りのエラー
   * 新たに導入されたエラー

注意:
- パッケージの復元が要る場合(`NU1101` 等)は `dotnet restore` を先に試す。
- フローティングバージョン(`9.*-*` 等)で API 名が違う場合は、現行パッケージのシンボルに読み替える。
- 安全のため、一度に 1 つのエラーのみを修正してください。
