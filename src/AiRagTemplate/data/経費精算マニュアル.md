# 経費精算マニュアル(抜粋) — 架空企業サンプル

本文書は RAG サンプル用の架空企業「サンプル合同会社」の経費精算マニュアル抜粋です。実在の規程ではありません。

## 1. 締め日と支払日

経費の締め日は毎月末日とする。当月末日までに申請された経費は、翌月25日に給与口座へ振り込む。

月末が休日の場合は前営業日を締め日とする。締め日を過ぎた申請は翌月分として扱う。

## 2. 申請の手順

経費精算は経費精算システム「Sample Expense」から行う。領収書またはインボイス要件を満たす適格請求書を添付すること。

申請は支出日から30日以内に行うこと。30日を超えた申請は原則として承認されないが、やむを得ない事情がある場合は上長の補足コメントを添えて申請する。

1件あたり1万円を超える支出は、事前に稟議で承認を得ること。事前承認のない高額支出は精算できない場合がある。

## 3. 交通費

公共交通機関を利用した場合は実費を精算する。IC カードの利用履歴または経路検索結果を証憑として添付する。

タクシーの利用は、深夜時間帯(午後10時以降)または重い荷物の運搬を伴う場合に限り認める。利用理由を申請時に記載すること。

自家用車での出張は、1キロメートルあたり15円のガソリン代相当を支給する。高速道路料金は実費精算とする。

## 4. 出張・宿泊

宿泊費の上限は1泊あたり国内1万2千円、海外1万8千円とする。上限を超える場合は差額を自己負担とする。

出張日当は国内出張1日あたり2千円を支給する。日帰り出張には日当を支給しない。

## 5. 精算できない費用

私的な飲食、罰金・反則金、個人的な書籍やサブスクリプションは精算の対象外とする。判断に迷う場合は経理部へ事前に確認すること。
